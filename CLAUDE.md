# 项目概述

## 目标
这是一个技术博客，主要发布一些技术文章，有时候会分享一些个人小感悟。

## 技术栈
- 框架：Hexo + Fluid 主题
- 部署：只需push代码到github，即可部署到 vercel。
- 写作方式：使用 hexo 命令行工具新建文章，或者在编辑器中直接编辑文章文件。

## 项目结构
- `source/_posts/` — 博客文章
- `themes/fluid` — 主题配置
- `db.json` — 文章数据库

## 技术范围
不限

## 文章风格
- 深度技术文章，注重实战
- 包含代码示例
- 语言：中文
- 如果能够用架构图来说明技术原理，会更方便理解

## 文章模板
---
title: {{title}}
date: {{date}}
tags: 根据文章内容打标签
categories: 根据文章内容分类
---

## 资料搜集
- 主要来源：官方文档、Web搜索
- 首选官方文档，因为我喜欢官方文档的严谨性和准确性


## 验证流程
1. 我整理好文章草稿
2. 展示给你确认
4. 展示的时候使用 hexo server -p 80 本地预览
5. 你确认后，我推送到 GitHub
