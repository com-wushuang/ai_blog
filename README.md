## 🚀 快速开始

### 1. 创建新文章

```bash
hexo new "文章标题"
# or
hexo n "文章标题"
```

### 2. 本地预览

```bash
hexo server -p 80
# 访问 http://localhost:80
```

### 3. 部署上线

```bash
git add .
git commit -m "update"
git push
# 自动部署到 Vercel
```
